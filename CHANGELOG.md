# Changelog

Todos los cambios notables de este proyecto se documentan en este archivo.

Formato basado en [Keep a Changelog](https://keepachangelog.com/es/1.0.0/),
y el proyecto sigue [Semantic Versioning](https://semver.org/lang/es/).

## [Sin publicar]

### Agregado
- Setup inicial del proyecto Angular 21 + Supabase
- Módulo de autenticación (login, registro, OAuth, profile-setup, callback)
- Grilla semanal de canchas (vista pública)
- Módulo de reservas (listado, nueva reserva, detalle)
- Registro de resultados de partidos
- Módulo de academia (clases y horarios)
- Perfil de jugador
- Panel de administración (dashboard, canchas, jugadores, academia, notificaciones)
- Layout responsive (top-nav, sidebar desktop, bottom-nav mobile)
- Integración con Supabase Auth y base de datos PostgreSQL
- Integración con Resend para notificaciones por email
- Soporte PWA
- Configuración Tailwind CSS v3 + Angular Material

## Tipos de cambios

- `Agregado` — nuevas funcionalidades
- `Modificado` — cambios en funcionalidades existentes
- `Obsoleto` — funcionalidades próximas a eliminarse
- `Eliminado` — funcionalidades eliminadas
- `Corregido` — corrección de bugs
- `Seguridad` — correcciones de vulnerabilidades
